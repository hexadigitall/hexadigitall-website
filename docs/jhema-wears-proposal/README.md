## üí° Best Practices

### Do's ‚úÖ
- **Personalize every message** - mention specific products you liked
- **Engage authentically** - like posts, comment genuinely
- **Be patient** - don't expect immediate responses
- **Follow up strategically** - 2-3 days, then 1 week
- **Offer value** - free guides, tips, resources
- **Be professional** - even in casual platforms
- **Track everything** - what works, what doesn't

### Don'ts ‚ùå
- **Don't spam** - quality over quantity
- **Don't be pushy** - respect their time
- **Don't send generic messages** - they'll ignore them
- **Don't criticize** - their current setup
- **Don't overwhelm** - with too much info at once
- **Don't give up** - after one no-response

## üìû Follow-Up Strategy

### Timeline

| Day | Action | Platform | Message Type |
|-----|--------|----------|--------------|
| 0 | Initial outreach | All | Introduction |
| 2-3 | Follow-up 1 | Same as initial | Friendly reminder |
| 7 | Follow-up 2 | Same as initial | Last message + value |
| 14 | Re-engage | Different platform | Fresh approach |

### Follow-Up Rules

1. **Maximum 3 follow-ups** per platform
2. **Wait at least 2 days** between messages
3. **Add value each time** (don't just ask again)
4. **Switch platforms** if no response after 3 attempts
5. **Know when to stop** - respect their silence
