## Generate/Update OG Assets

To regenerate or add assets, run:

```bash
# Generate service/course OG cards
node scripts/generate-og-images.mjs

# Generate course-specific OG cards from Sanity content
node scripts/generate-og-course-images.mjs
```

Assets are written to [public/og-images](public/og-images). Promo images are under [promotional-campaign/images/designed](promotional-campaign/images/designed).
