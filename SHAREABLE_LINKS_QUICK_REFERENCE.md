## ðŸ“± PLATFORM-SPECIFIC TIPS

### INSTAGRAM BEST PRACTICES
- Caption length: 100-150 characters (with line breaks)
- Use first comment for longer CTA
- Include 3-5 hashtags in first comment (not main caption)
- Stories: Use interactive stickers (polls, questions)
- Reels: Hook in first 3 seconds
- Post times: 9am, 12pm, 6pm, 7pm

### TIKTOK BEST PRACTICES
- First 3 seconds: Hook viewers
- Use trending audio (even if repurposed)
- Hashtags: 3-5 max
- Consistency > perfection
- Respond to comments with videos
- Duets/Stitches: Respond to trends

### FACEBOOK BEST PRACTICES
- Longer form content performs well
- Video auto-plays = higher reach
- Community engagement builds algorithm
- Post times: 1-3pm, 7-9pm
- Ask questions in posts
- Use carousel ads for multiple services

### LINKEDIN BEST PRACTICES
- Professional tone with personality
- Share insights/thoughts, not just promotions
- Engage with others' content first
- Use document posts for in-depth content
- Post times: Tuesday-Thursday, 9am-2pm
- Tag relevant people (not excessive)

### WHATSAPP BEST PRACTICES
- Use broadcast lists, not group chats
- Segment lists by interest
- Send 1-2 messages max per day
- Respect opt-out requests
- Personalize where possible
- Include clear CTA
